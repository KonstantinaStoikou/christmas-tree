## C program that outputs a christmas tree with random ornaments
You can change the LINES definition to increase or decrease the size of the tree.  
Example output for LINES 10:
```
         *
        ***
       O**O*
      ***O***
     *********
    ***********
   O*********O**
  ***O*****O*****
 *******O*********
********O********O*
      ######
      ######
      ######
      ######
      ######
```
